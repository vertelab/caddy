# caddy
Caddy scripts
| Command              | Usage                        | Description                                         |
| -------------------- | ---------------------------- | --------------------------------------------------- |
| caddy-ensite         | caddy-ensite                 | List available sites in /etc/caddy/sites-available/ |
| caddy-ensite <site>  | caddy-ensite odoo19.conf  | Enable site (create symlink to sites-enabled/)      |
| caddy-dissite        | caddy-dissite                | List enabled sites in /etc/caddy/sites-enabled/     |
| caddy-dissite <site> | caddy-dissite odoo19.conf | Disable site (remove symlink)                       |
